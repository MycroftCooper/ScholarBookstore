# 首页 & 发现模块需求文档

首页、发现页——内容入口与聚合。

---

## 1. 模块概述

| 页面 | 路由 | 说明 |
| --- | --- | --- |
| 首页 | `/` | 站点入口，版块导航 + 最新文章 |
| 发现页 | `/discover` | 搜索 + 随机文章 + 热点聚合 |

---

## 2. 首页 `/`

### 2.1 展示内容

**区域 1：顶部导航**
- Logo + 站点名
- 领域导航入口（下拉或平铺）
- 搜索入口（跳转 `/discover`）
- 已登录：用户头像 + 通知入口（含未读数徽章）+ 个人中心入口
- 未登录：登录/注册入口

**区域 2：版块入口**
- 展示所有启用的版块列表（可按所属领域分组展示）
- 简化版：展示版块名称卡片，点击进入 `/modules/[slug]`

**区域 3：最新文章**
- 按发布时间倒序展示最近 N 篇 `published` 文章
- 文章卡片展示：标题、摘要、作者头像+用户名、发布时间、字数、浏览量、Tag

### 2.2 页面状态

| 状态 | 展示 |
| --- | --- |
| 正常 | 版块入口 + 最新文章列表 |
| 无版块 | 版块区展示"暂无版块" |
| 无文章 | 文章区展示"暂无文章" |
| 加载中 | 骨架屏 |
| 加载失败 | 错误提示 + 重试 |

### 2.3 移动端

- 版块和文章列表单列展示
- 导航收敛为移动端菜单（汉堡菜单）

---

## 3. 发现页 `/discover`

### 3.1 功能分区

发现页包含三个功能区域：

| 区域 | 功能 |
| --- | --- |
| 搜索 | 全文搜索已发布文章 |
| 热点聚合 | 近期热门文章（浏览量 + 时间衰减加权） |
| 随机推荐 | 随机展示几篇已发布文章 |

### 3.2 搜索功能

**搜索范围**：
- 仅已发布文章
- 搜索字段：标题 + 摘要 + 正文
- 支持按 Tag 筛选

**搜索方式**：
- 关键词输入（PostgreSQL 全文搜索）
- Tag 筛选（点击 Tag 或从列表选择）
- 两者可组合使用

**搜索结果**：
- 分页列表
- 每项展示：标题、摘要、作者、发布时间、Tag
- 按相关度排序

**搜索结果为空时**：
- 展示"未找到相关文章"提示
- 建议尝试其他关键词

### 3.3 热点聚合

**排序规则**：
- 按浏览量 + 发布时间衰减加权排序
- 时间衰减：近期文章权重更高
- 算法示例：`score = views / (hours_since_publish + 1)^0.8`

**展示**：
- 热点文章列表（标题 + 浏览量 + Tag）
- TODO：具体展示条数和样式后续确认

### 3.4 随机推荐

**规则**：
- 从已发布文章中随机选取 N 篇（如 5 篇）
- 每次刷新/访问可能不同

**展示**：
- 随机文章卡片列表

### 3.5 页面状态（整体）

| 状态 | 展示 |
| --- | --- |
| 正常 | 搜索栏 + 热点 + 随机（可切换 tab 或分区展示） |
| 搜索中 | 搜索区域 loading |
| 搜索结果为空 | "未找到相关文章" |
| 搜索错误 | 错误提示 + 重试 |
| 加载失败 | 对应区域错误提示 + 重试 |

### 3.6 移动端

- 搜索栏置顶
- 热点和随机分区可折叠或 tab 切换

---

## 4. 搜索入口

搜索可从以下位置触发：

1. 顶部导航栏搜索图标 → 跳转 `/discover`，自动聚焦搜索框
2. 文章 Tag 点击 → 跳转 `/discover?tag=xxx`
3. 直接访问 `/discover`
4. 版块页或首页的搜索入口

---

## 5. 权限约束

- 首页和发现页对所有用户（含游客）开放
- 搜索仅返回 `published` 且未删除文章
- 游客不可见文章评论（即使从发现页进入文章详情）
